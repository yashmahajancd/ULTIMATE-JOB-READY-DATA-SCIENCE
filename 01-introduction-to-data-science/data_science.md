## What is Data Science?

*Data Science* is the process of *collecting, cleaning, analyzing, and interpreting data* to make informed decisions.

It combines:
- Mathematics & Statistics
- Programming
- Domain knowledge
- Problem-solving mindset

Rather than being purely theoretical, Data Science focuses on *solving real-world problems using data*.

## Data Science Lifecycle

The Data Science Lifecycle is a structured and iterative process:

### 1. Problem Definition
- Understand the business problem
- Define objectives and success metrics

### 2. Data Collection
- Gather data from databases, APIs, web scraping, or external sources

### 3. Data Cleaning (Preprocessing)
- Handle missing values
- Remove duplicates
- Fix inconsistencies

### 4. Exploratory Data Analysis (EDA)
- Analyze distributions
- Visualize patterns
- Identify correlations and outliers

### 5. Model Building
- Apply machine learning algorithms
- Train models on historical data

### 6. Model Evaluation
- Measure performance using metrics  
  (Accuracy, Precision, Recall, RMSE, etc.)

### 7. Deployment
- Integrate models into real systems
- Use APIs or web applications

### 8. Communication & Reporting
- Present insights clearly to stakeholders
- Create dashboards and reports

### 9. Maintenance & Iteration
- Monitor model performance
- Retrain models with new data
