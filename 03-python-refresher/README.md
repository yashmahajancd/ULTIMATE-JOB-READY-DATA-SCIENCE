# Python Refresher for Data Science

This document provides a *concise refresher of Python fundamentals* required for Data Science and Machine Learning.  
The focus is on *practical usage*, clean syntax, and concepts that are heavily used in real-world data workflows.

---

## Why Python for Data Science?

Python is the preferred language in Data Science because it:
- Is easy to read and write
- Has a massive ecosystem of data libraries
- Supports rapid experimentation
- Scales from simple scripts to large ML pipelines

A strong Python foundation is essential before moving to NumPy, Pandas, and Machine Learning.
